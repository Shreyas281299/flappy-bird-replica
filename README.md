# Flappy Bird Replica

A small Flappy Bird-style game built with Pygame. The game uses simple drawn
background elements plus downloaded bird and pipe sprites.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Controls

- Press `Space`, `Up Arrow`, or click to flap.
- Press `R` after a game over to restart.
- Press `Esc` to quit.

## Assets

The bird and pipe PNG sprites are downloaded from the online
`samuelcust/flappy-bird-assets` sprite set.
